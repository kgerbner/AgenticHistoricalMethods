✍️ Transcribing a difficult source

Use Claude to get a first pass on hard handwriting or degraded print, then verify line by line.

  1. Photograph or scan the source at high resolution. If using OCR first, paste the raw OCR text too.
  2. Give Claude context: date, language, hand/typeface, and any known abbreviations.
  3. Ask for a transcription that preserves original spelling and brackets uncertain readings.
  4. Open the image beside the output and correct every bracketed and suspicious passage.
  5. Save both the corrected transcription and a note of what Claude got wrong — it's good methods documentation.

Prompt

I'm transcribing a [1680s English manuscript, secretary hand].
Here is an image description / OCR of the page: [paste]

Transcribe it, preserving original spelling and punctuation.
- Put any uncertain reading in [brackets] with a one-word note.
- List every abbreviation you expanded.
- Do NOT silently guess; flag low-confidence lines.

✦ Never publish a transcription you haven't checked against the original image yourself.

🔎 Analyzing a set of primary sources

Turn a pile of documents into structured, searchable data you can interrogate.

  1. Decide the variables you care about (e.g. date, place, people named, themes).
  2. Paste one source and ask Claude to extract those fields into a table.
  3. Refine the schema, then repeat across your set — or ask for a script to batch it.
  4. Spot-check rows against the sources; fix systematic errors by tightening the prompt.
  5. Ask Claude to summarize patterns as hypotheses to test, not conclusions.

Prompt

Extract the following from this source into a Markdown table:
| date | place | people named | document type | key themes |

Source: [paste]

If a field is absent, write "n/a". Quote names exactly as written.
Flag anything ambiguous in a "notes" column.

📚 Synthesizing secondary literature

Map a historiographical conversation from readings you supply — not from the model's memory.

  1. Gather the texts (PDFs, your notes, or pasted passages). Provide them to Claude.
  2. Ask it to summarize each author's argument and intervention in 2–3 sentences.
  3. Ask it to map where scholars agree, disagree, and talk past each other.
  4. Draft a literature-review paragraph — then rewrite in your voice and re-check every citation.

Prompt

Here are summaries/excerpts of 5 works on [topic]: [paste]

1. State each author's central argument and what they're arguing against.
2. Group them into the main positions in the debate.
3. Note tensions and unanswered questions.
Use ONLY the texts I provided. If you're missing context, say so.

✦ Verify every author, date, and quotation before any of this reaches a draft.
